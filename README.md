### Getting Started

Clone this repo:

```bash
git clone https://github.com/arcmena/pokedexnextjs.git
```

Install the dependencies:

```bash
npm install
```

or simply:

```bash
yarn
```

And run to start:

```bash
yarn dev
```

> **Note:** The server will start at http://localhost:3000 by default.
